# 六神象意冲突表

## 原则

六神只作象意补充，不得推翻用神、世应、月日、动变、生克制化的主结构。

## 常见冲突

| 六神象 | 主结构 | 处理 |
|---|---|---|
| 青龙临忌神 | 忌神旺动克用 | 不可简单断吉，只能说表面有好象或诱因 |
| 白虎临原神 | 原神有力生用 | 不可简单断凶，白虎只提示压力、损伤、急迫性 |
| 玄武临用神 | 用神旺相 | 可提示隐情、暗处、拖延，但不能直接断坏 |
| 朱雀临父母 | 父母为文书 | 可提示信息、口舌、消息、材料，但需看旺衰 |
| 腾蛇临动爻 | 动爻为关键 | 可提示纠缠、虚惊、反复，但不替代动变生克 |
